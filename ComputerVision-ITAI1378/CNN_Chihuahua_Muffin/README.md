# CNN Image Classifier: Chihuahua vs. Muffin

## Problem Statement
Build a CNN that accurately distinguishes between images of chihuahuas and muffins.

## Approach
* Built and trained a CNN model using TensorFlow/Keras
* * Applied data augmentation to improve model generalization
  * * Tuned hyperparameters including learning rate, batch size, and layer depth
   
    * ## Results
    * * Achieved high accuracy on the binary classification task
      * * Generated training/validation accuracy and loss curve visualizations
       
        * ## Key Findings
        * * Data augmentation significantly reduced overfitting on the small dataset
          * * Deeper networks with dropout regularization outperformed simpler architectures
           
            * ## Technologies Used
            * * Python 3, TensorFlow / Keras, NumPy, Matplotlib, Jupyter Notebook
             
              * ## How to Run
              * Open `Lab05_CNN_Chihuahua_Muffin.ipynb` in Google Colab or run locally with Python 3.8+.
