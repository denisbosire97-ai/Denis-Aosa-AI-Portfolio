# Computer Vision (ITAI 1378) - Course Overview

> Houston Community College | Spring 2026
>
> ## Course Description
> This course covers fundamental and advanced computer vision techniques, from classical image processing to deep learning-based object detection.
>
> ## Projects
>
> | Project | Description | Key Technologies |
> |---------|-------------|-----------------|
> | [Safety Coordinator Glasses](Project1-Safetycoordinator-Glasses/README.md) | Real-time edge detection for worker safety | YOLOv8-Nano, MediaPipe, OpenCV |
> | [CNN Chihuahua vs. Muffin Classifier](CNN_Chihuahua_Muffin/README.md) | Binary image classification with CNNs | TensorFlow, Keras, Matplotlib |
> | [Object Detection Labs (L06, L08, L11)](Object_Detection_Labs/README.md) | Structured labs on detection techniques | OpenCV, PyTorch, Jupyter |
>
> ## Skills Developed
> - Convolutional Neural Network (CNN) architecture design and training
> - - Real-time object and pose detection using pre-trained models
>   - - Edge deployment considerations for AI vision systems
>     - - Dataset preparation, augmentation, and evaluation metrics
